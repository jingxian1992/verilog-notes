案例代码

// 定义参数
parameter latency = 20;
parameter delta = 2;

// 定义寄存器变量
reg x, y, z, p, q;

initial
begin
	x = 0;    // 没有延迟控制
	#10 y = 1;   // 延迟是数字的延迟控制。第10个时间单位才执行 y = 1
	#latency z = 0;  // 使用标识符的延迟控制。延迟20个时间单位
	#(latency + delta) p = 1;   // 使用表达式控制的延迟
	
	#y x = x + 1;  // 使用标识符的延迟控制，用 y 的值。
	
	#(4:5:6) q = 0;  // 最小、典型、最大延迟。
end

这个案例代码很有用。其中呢，行号19里面的东西，是我这里要去讨论的。
从第19行可以了解到，【最小、典型、最大延迟】的知识，不但是可以用于
门延迟，也可以用于过程赋值语句。
